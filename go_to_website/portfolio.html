Three A/B tests you would run on or website to improve the user experience.

A/B test one – customer value propositions.

Use Facebook ads to run a campaign with two ad sets - one designed to target consumers looking for hydrating anti-ageing products, the other designed to target consumers seeking simple, easy to use cleansing products.

Then create two landing pages, each with a tailored message to reflect the campaign assets.

Message One- Glow-up, hydrate, nourish your skin.

Message Two - Uncomplicated, effective, worry-free.

A/B test two – CTAs.

Test two alternative hero images on the landing page, each with different calls to action.

Message One – Double Cleanse. So Clean. So Soft. SHOP NOW.

Message Two – Vitamin A.B.C, what does your skin need? TAKE THE SKIN QUIZ.

*leads to questionnaire about skin type which recommends products.
